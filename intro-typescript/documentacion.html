TypeScript

Es un super set de javaScript Desarrollado por microsoft, del 85% al 90% es javaScript nativo, y añade nuevos estandares de ECMAScript 6 y 7, posee un tipado fuerte, es decir que se puede indicar el tipo de dato que contiene una variable, una programacion orientada a objetos mejorada, es un lenguaje interpretado que se transpila a javaScript. Es decir que codificamos en TypeScript y el transpilador que posee lo transforma o lo transpila a javaScript, para que todos los navegadores puedan entender. Y la extencion de los archivos es (.ts). Ademas este es el lenguaje nativo para trabajar en Angular.

tsc -v Para ver la version

tsc nombreArchivo.ts para que los transpile

tsc -w *.ts Para que este pendiente de todos los archivos y los campos

// Clase (Es el molde del objeto) class Camiseta{ // Propiedades (Caracteristicas del objeto) /** * Si la propoedades esta public cualquier instancia de * la clase las puede modificar * * Si esta private solo los metodos de la dicha clase puede modificarlos * * * */ private color: string; private modelo: string; private marca: string; private talla: string; private precio: number; // Metodos (que son funciones o acciones que tendra el objeto) // consturctor constructor(color, modelo, marca, talla, precio){ this.marca = marca; this.modelo = modelo; this.color = color; this.talla = talla; this.precio = precio; } /** * Metodo para cambiar el color * setColor */ public setColor(color) { this.color = color; } /** * getColor */ public getColor() { return this.color; } } var camiseta = new Camiseta("Rojo", "manga larga","nike", "L", 14); // camiseta.setColor("rojo"); // camiseta.getColor(); var plallera = new Camiseta("Azul", "Manga corta", "addidas", "M", 20); plallera.setColor("Azul"); // camiseta.getColor(); console.log(camiseta.getColor(), plallera.getColor());

Decoradores

un decorador es un patron de diseño que nos permite mediante unos metadato que nosotros le definimos de acuerdo a una clase, hacer una copia de esa misma clase y modificarlas y que haga unas cosas u otras dependiendo los parametros que les pasemos.
